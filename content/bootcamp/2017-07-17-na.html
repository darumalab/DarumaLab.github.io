ベクトル

x <- NA
num <- c(1, 2, 3, NA)
str <- c("dog", "cat", NA, "bee")
logi <- c(TRUE, NA, FALSE, TRUE)

データのタイプ

mode(num)
## [1] "numeric"
mode(str)
## [1] "character"
mode(logi)
## [1] "logical"

NA(欠損値)の検出

is.na(num)  #または!is.na(num)
## [1] FALSE FALSE FALSE  TRUE
is.na(str)
## [1] FALSE FALSE  TRUE FALSE
is.na(logi)
## [1] FALSE  TRUE FALSE FALSE

NAのインデックス:

which(is.na(num))  #which()でTRUEのインデックスを検出
## [1] 4
which(is.na(str))
## [1] 3
which(is.na(logi))
## [1] 2

ベクトルからNAを取り除く

num <- num[!is.na(num)]
